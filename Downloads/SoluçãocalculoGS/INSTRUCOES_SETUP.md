#  Instrucoes para Rodar

# Pre-requisitos

- **Node.js 18+** (download em https://nodejs.org/)
- **pnpm** (gerenciador de pacotes - mais rapido que npm)

## Passo 1: Instalar o pnpm (se nao tiver)

```bash
npm install -g pnpm
```

Verificar instalação:
```bash
pnpm --version
```

##  Passo 2: Clonar/Baixar o Projeto

Se voce recebeu um arquivo ZIP:
```bash
unzip ecoalert-front.zip
cd ecoalert-front
```

##  Passo 3: Instalar Dependencias

```bash
pnpm install
```

##  Passo 4: Rodar o Servidor de Desenvolvimento

```bash
pnpm dev
```

Voce vera algo como:
```
 Local:   http://localhost:3000/
```

##  Passo 5: Abrir no Navegador

Abra seu navegador e acesse:
```
http://localhost:3000
```
